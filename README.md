# Frontend Mentor Practice Projects

A portfolio repository for Frontend Mentor HTML/CSS/JS challenge solutions.

## Quick Start

Open the root `index.html` page (project launcher) to jump to each challenge:

- `blog-preview-card-main/index.html`
- `qr-code-component-main/index.html`
- `social-links-profile-main/index.html`
- `faq-accordion-main/index.html`
- `browser-extensions-manager-ui-main/index.html`

## How to run

1. Open this repository in VS Code.
2. Open `index.html` in the workspace root.
3. Use Live Server for instant preview and refresh while editing.

## Projects

### 1) Blog Preview Card

- Folder: `blog-preview-card-main`
- Stack: HTML, CSS
- Goal: Recreate the card layout with responsive spacing and hover states.

### 2) QR Code Component

- Folder: `qr-code-component-main`
- Stack: HTML, CSS
- Goal: Build a centered card with a QR image and text styling.

### 3) Social Links Profile

- Folder: `social-links-profile-main`
- Stack: HTML, CSS
- Goal: Build a profile card with social media links and hover/focus states.

### 4) FAQ Accordion

- Folder: `faq-accordion-main`
- Stack: HTML, CSS, JavaScript
- Goal: Build an accessible accordion that shows/hides answers on click with full keyboard navigation.

### 5) Browser Extensions Manager

- Folder: `browser-extensions-manager-ui-main`
- Stack: HTML, CSS, JavaScript
- Goal: Build a UI to toggle, filter and remove extensions with a theme switcher and dynamic JSON data.

## Notes

- This repo is for learning and portfolio practice.
- To add a new project: create the folder, build the challenge, then link it from the root `index.html` and update this README.
